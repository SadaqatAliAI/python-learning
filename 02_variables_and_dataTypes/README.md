# Variables and Data Types

This folder contains beginner-friendly Python programs that explain the fundamental concepts of **variables** and **built-in data types**. These topics form the foundation of Python programming and are essential before moving on to more advanced concepts.

## 📂 Contents

| File | Description |
|------|-------------|
| `Local and Global Variable.py` | Demonstrates the difference between local and global variables, their scope, and how they are used in functions. |
| `Integers.py` | Explains integer data types and basic integer operations. |
| `Strings.py` | Introduces strings, string creation, indexing, slicing, and common operations. |
| `String Methods.py` | Covers commonly used built-in string methods for manipulating text. |
| `Float.py` | Explains floating-point numbers and arithmetic with decimal values. |
| `Boolean.py` | Introduces Boolean values (`True` and `False`) and their role in logical operations. |
| `List.py` | Demonstrates Python lists, indexing, slicing, and common list operations. |
| `Tuple.py` | Explains tuples, their immutability, and how they differ from lists. |
| `Dictionary.py` | Covers dictionaries, key-value pairs, and common dictionary operations. |
| `Set.py` | Introduces sets, unique elements, and set operations. |
| `None.py` | Explains the `None` object and when it is used in Python. |
| `Complex.py` | Introduces complex numbers and basic operations with them. |

## 🎯 Learning Objectives

After completing these examples, you will be able to:

- Understand what variables are and how to use them.
- Differentiate between local and global variables.
- Work with Python's built-in data types.
- Perform basic operations on integers, floats, and complex numbers.
- Manipulate strings using common string methods.
- Use Boolean values in logical expressions.
- Store and manage data using lists, tuples, dictionaries, and sets.
- Understand the purpose of the `None` value.

## 🚀 Prerequisites

- Python 3.x installed on your system.
- A code editor such as Visual Studio Code, PyCharm, or IDLE.
- Basic understanding of how to run Python programs.

## ▶️ Running the Examples

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

Navigate to this folder:

```bash
cd "Variables and Data Types"
```

Run any file:

```bash
python filename.py
```

Example:

```bash
python Strings.py
```
